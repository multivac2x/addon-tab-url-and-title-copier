🧪 Supabase Connection Tester

Purpose: This tool helps you verify that your Supabase configuration is correct before using it in the Firefox extension.
Before you start:
  1. Create a Supabase project at supabase.com
  2. Run the SQL commands from supabase-setup.sql in your Supabase SQL editor
  3. Enable Anonymous authentication in your Supabase project settings
  4. Get your project URL and anon key from the API settings

Enter Your Supabase Configuration

Next Steps: Once the connection test passes, copy your URL and key to the supabase-config.js file in your extension directory.
